# Хаос в музыке

Исследовательский проект, в котором музыкальные аудиосигналы 
анализируются как временные ряды с точки зрения теории 
динамического хаоса и нелинейной динамики.

## Описание

В работе рассматриваются три задачи:
- классификация жанров инструментальной музыки
- детекция сгенерированных нейросетью инструментальных произведений
- детекция сгенерированных нейросетью вокальных композиций

Используются три метода: плоскость энтропия-сложность, 
алгоритм Швайнхарта и алгоритм MLE.

## Датасеты

- [GTZAN](https://www.kaggle.com/datasets/andradaolteanu/gtzan-dataset-music-genre-classification)
- [FakeMusicCaps](https://arxiv.org/abs/2409.10684)
- [SONICS](https://arxiv.org/abs/2408.14080)

## Структура репозитория

- `validation.ipynb` — валидация методов на синтетических объектах
- `genres.ipynb` - классификация жанров инструментальной музыки
- `classical.ipynb` — детекция сгенерированных нейросетью инструментальных произведений
- `sonics.ipynb` — детекция сгенерированных вокальных композиций
